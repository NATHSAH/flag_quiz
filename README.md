#  Flag Quiz App

A web-based quiz application where users guess countries based on their flags.

## Features

* 🌍 All country flags included
* 🎯 Score tracking system
* 🔤 Case-insensitive answers
* ⚡ Smooth gameplay (Enter to submit & next)
* 🎨 Clean and simple UI

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS, JavaScript
* REST Countries API / Local Dataset

## ▶️ How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/NATHSAH/flag_quiz.git
   ```

2. Navigate to the project:

   ```bash
   cd flag_quiz
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the app:

   ```bash
   python app.py
   ```

5. Open in browser:

   ```
   http://127.0.0.1:5000
   ```
